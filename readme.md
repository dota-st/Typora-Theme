![image-20220310200352134](readme.assets/image-20220310200352134.png)

## 使用方式

1. 文件-> 偏好设置 -> 外观 -> 打开主题文件夹
2. 下载后，把`dotast`文件夹和`dotast.css`放到该文件夹下
3. 重启typora，食用
